# Audio Files Directory

This directory contains background ambient music for the portfolio website.

## Required File

Place your audio file here:
```
ambient-koto.mp3
```

## File Specifications

- **Format:** MP3
- **Style:** Japanese Koto-inspired ambient
- **Tempo:** 60-80 BPM (slow, calming)
- **Volume:** Will play at 12% (configurable)
- **Duration:** 2-5 minutes (will loop seamlessly)
- **File Size:** Under 5MB (ideally 2-3MB)
- **No vocals**

## Where to Get Music

See the main `AUDIO-SETUP.md` file in the project root for:
- Free music sources
- Premium options
- Licensing information
- Detailed setup instructions

## Testing Without Audio

The app will function normally without this file:
- Floating music button will appear
- UI will be interactive
- Console will show a warning
- No sound will play
